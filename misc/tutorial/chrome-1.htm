Windows/Mac/Linux 使用 Free-server 的 HTTP2 设置 — Get on Facebook in China

安装步骤

1. 安装 Chrome 浏览器 —— 如果你已经有Chrome，请删除之；或者确保其版本是V46以上，要不可能安装不上新版本 （官方离线安装包中国大陆下载, Windows / Mac 苹果 都有，每周自动更新）

3.（看下面截图）下载 SwitchyOmega_Chromium 自动代理切换插件：

3.1 打开你的 Chrome 浏览器，最大化 Chrome 窗口

3.2 打开你的邮箱（qq、163、126邮箱？），并且关闭其他浏览器（比如你用 360 浏览器）

3.2 找到 Free-Server 发给你的那封邮件

3.3 在邮件中打开这个教程（或者直接拷贝此页面的链接到 Chrome，如果你知道如何拷贝链接的话），然后再做下面剩下的步骤。

3.4 使用Chrome浏览器点击这个下载链接： SwitchyOmega_Chromium，下载结束后，点击 “在文件夹中显示”。

 

4.（看下面截图）将这个zip包解压缩；得到文件夹 SwitchyOmega_Chromium

 

5. 看这个视频教程

注意，在视频教程中出现的 点击 “导入/导出” 标签，你需要从这个链接导入，而不是视频中的

__SERVER_URL__/tutorial/OmegaOptions.bak.html （点击即可拷贝）

如果导入失败，你就再多点击几次 “恢复” 即可。

 

6.（看下面截图）准备启用Fan墙模式～～点击 Chrome 右上角的黑色小圆圈，就会弹出 Fan 墙选项选择下拉菜单。然后选择菜单中的： “快速： SPDY智能Fan墙”

为了避免你将来不小心动了这个插件，建议你最好隐藏它：

7. 关闭所有的 Chrome 浏览器，重启 Chrome；然后打开 Facebook 看，可以了吧。

如果仍无法翻 qiang 或者出现下图字样，可能是因为你输入的Fan墙账号信息有误，请重复检查一下 第6-10步里面的代理服务器(默认是 server1.free-server.me，但这往往与你的服务器不相同)、代理端口、用户名、密码是否与你得到的信息一样，并且别忘了第11步的应用选项。

 

还没结束哦：现在应该是可以上 Facebook 的， 有图有真相：

 

8. 如果你在第一次成功翻墙之后，还偶尔出现如下的需要登录 SPDY 代理服务器的对话框，你有两个选择：

  • 选择1： 什么都不做，直接无视那个登录框。同时新建一个标签页，该干吗干吗。这个其实是个bug，将来应该会修复。
  • 选择2：请输入你的 SPDY 用户名与密码，并点击登录即可。然后 Chrome 可能会弹出保密密码提示，点击保存即可。再弹出来，你就可以直接点击登录了～

 

 

9. 如果将来 Chrome 弹出翻墙代理插件警报提示，选择“保留更改”即可。

 

 

常见问题：

1. 为什么有时候即使Fan墙了，某些网站还是没有办法打开呢？

答：我们的“智能Fan墙”可能还没有收录你正在访问的网站，你可以尝试使用 慢速：HTTP2/SPDY 全部Fan墙。如果还不行，就说明那个网站已经不能访问了。我们不太推荐“全部Fan墙”方案，因为这会导致我们的服务器流量徒然增大，也可能会导致你无法打开或者慢速打开国内的网站。
