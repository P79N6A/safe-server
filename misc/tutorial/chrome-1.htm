Windows/Mac/Linux 使用守望无墙计划的 HTTP2 设置 — Get on Facebook in China

安装步骤

1. 安装 Chrome 浏览器 —— 如果你已经有Chrome，请删除之；或者确保其版本是V46以上，要不可能安装不上新版本 （官方离线安装包中国大陆下载, Windows / Mac 苹果 都有，每周自动更新）

3.（看下面截图）下载 SwitchyOmega 自动代理切换插件：

3.1 打开你的 Chrome 浏览器，最大化 Chrome 窗口

3.2 打开你的邮箱（qq、163、126邮箱？），并且关闭其他浏览器（比如你用 360 浏览器）

3.2 找到守望无墙计划的那封邮件

3.3 在邮件中打开这个教程（或者直接拷贝此页面的链接到 Chrome，如果你知道如何拷贝链接的话），然后再做下面剩下的步骤。

3.4 使用Chrome浏览器点击这个下载链接： SwitchyOmega，下载结束后，点击 “在文件夹中显示”。（如何点击？看下面截图。）

 

4.（看下面截图）将之重命名为 SwitchyOmega.crx (删除后面的所有扩展名，比如.sw，使之成为 SwitchyOmega.crx)，按回车确定

 

5.（看下面截图）使用 Chrome 浏览器，点击右上角的菜单按钮打开 其扩展程序页面

 

 

5.1（看下面截图）并将刚才重命名的那个文件拖放到 你刚打开的 Chrome 扩展程序 的正中间位置，松开鼠标，然后点击添加。

 

（看下面截图）然后等2秒，再点击弹出来的添加

 

6.（看下面截图）进入插件设置：

6.1 点击右上角的小圆圈；然后在弹出的菜单中，选择“选项”，进入“插件设置”

6.2 （如果）出现 “欢迎使用 SwitchyOmega” 字样的教程页面，则请点击 “跳过教程”，

6.2.1 “刷新” SwitchyOmega 页面（刷新按钮在Chrome浏览器左上角，那个顺时针的环形箭头——在前进、后退剪头的右边）。

 

7.（看下面截图）点击左边的 “导入/导出”，然后在 “在线恢复” 下输入：__SERVER_URL__/tutorial/OmegaOptions.bak.html，并点击 “恢复”。稍等片刻，你应该可以看到 “导入选项成功”，或者只要页面左边出现了“慢速： HTTP2/SPDY。。。”等字样即可。

如果导入失败，你就再多点击几次 “恢复” 即可。

 

8.（看下面截图）点击左边的 “慢速： SPDY 全部Fan墙” 选项，然后输入你得到的服务器地址和代理端口，然后点击那个密码图标

 

9.（看下面截图）输入用户名与密码，然后保存更改。

10.（看下面截图）最后别忘了点击 “应用选项”

11.（看下面截图）准备启用Fan墙模式～～点击 Chrome 右上角的黑色小圆圈，就会弹出 Fan 墙选项选择下拉菜单。然后选择菜单中的： “快速： SPDY智能Fan墙”

为了避免你将来不小心动了这个插件，建议你最好隐藏它：

12. 关闭所有的 Chrome 浏览器，重启 Chrome；然后打开 Facebook 看，可以了吧。

如果仍无法翻 qiang 或者出现下图字样，可能是因为你输入的Fan墙账号信息有误，请重复检查一下 第6-10步里面的代理服务器(默认是 server1.free-server.me，但这往往与你的服务器不相同)、代理端口、用户名、密码是否与你得到的信息一样，并且别忘了第11步的应用选项。

 

13. 还没结束哦：现在应该是可以上 Facebook 的， 有图有真相：

 

14. 如果你在第一次成功翻墙之后，还偶尔出现如下的需要登录 SPDY 代理服务器的对话框，你有两个选择：

  • 选择1： 什么都不做，直接无视那个登录框。同时新建一个标签页，该干吗干吗。这个其实是个bug，将来应该会修复。
  • 选择2：请输入你的 SPDY 用户名与密码，并点击登录即可。然后 Chrome 可能会弹出保密密码提示，点击保存即可。再弹出来，你就可以直接点击登录了～

 

 

15. 如果将来 Chrome 弹出翻墙代理插件警报提示，选择“保留更改”即可。

 

 

常见问题：

1. 为什么有时候即使Fan墙了，某些网站还是没有办法打开呢？

答：我们的“智能Fan墙”可能还没有收录你正在访问的网站，你可以尝试使用 慢速：HTTP2/SPDY 全部Fan墙。如果还不行，就说明那个网站已经不能访问了。我们不太推荐“全部Fan墙”方案，因为这会导致我们的服务器流量徒然增大，也可能会导致你无法打开或者慢速打开国内的网站。
